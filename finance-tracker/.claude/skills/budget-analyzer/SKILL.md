## Description

월간/연간 예산을 설정하고, 실제 지출과 비교하여 재무 상태를 분석합니다.

## When to Use

- 월간 예산을 설정할 때
- 예산 대비 지출 현황을 확인할 때
- 예산 초과 알림이 필요할 때
- 재무 목표 달성도를 추적할 때

## Features

### 1. 예산 설정
- 카테고리별 예산 할당
- 월간/연간 예산 계획
- 예산 조정 및 최적화

### 2. 실시간 모니터링
- 예산 사용률 추적
- 초과 지출 경고
- 잔여 예산 계산

### 3. 분석 리포트
- 예산 대비 실제 지출 비교
- 절약 기회 식별
- 지출 트렌드 분석

## Alerts

- 🔴 예산 80% 사용 시 경고
- 🟡 예산 100% 도달
- 🟢 예산 내 성공적 관리
- 📊 월말 종합 리포트

## Tips

- 50/30/20 규칙 활용 (필수 50%, 선택 30%, 저축 20%)
- 예산은 유연하게 조정
- 비상금 별도 관리
- 정기적인 예산 리뷰
