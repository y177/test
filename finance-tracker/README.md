# 💰 Finance Tracker

개인 재무를 추적하고 예산을 관리하는 도구입니다.

## ✨ 주요 기능

- 💸 **지출 추적**: 카테고리별 지출 기록 및 관리
- 💵 **수입 관리**: 다양한 소득원 추적
- 📊 **예산 관리**: 월간/연간 예산 설정 및 모니터링
- 📈 **통계 및 리포트**: 시각적 재무 분석
- 💾 **데이터 내보내기**: CSV, JSON 형식 지원

## 🚀 시작하기

### 1. 설치

```bash
npm install
```

### 2. 초기 설정

```bash
npm start
```

첫 실행 시 데이터베이스가 자동으로 생성됩니다.

### 3. 환경 변수 설정

`.env` 파일 생성:

```env
DATA_DIR=./data
REPORTS_DIR=./reports
CURRENCY=KRW
```

## 📖 사용 방법

### 지출 추가

```bash
npm run add-expense
```

대화형 인터페이스로 지출 정보 입력:
- 금액
- 카테고리
- 날짜
- 설명
- 결제 수단

### 수입 추가

```bash
npm run add-income
```

### 예산 관리

```bash
npm run budget
```

카테고리별 예산 설정 및 현황 확인

### 리포트 생성

```bash
npm run report
```

월간/연간 재무 리포트 생성

### 통계 보기

```bash
npm run stats
```

## 💡 재무 관리 팁

### 50/30/20 규칙
- **50%**: 필수 지출 (주거, 식비, 공과금)
- **30%**: 선택 지출 (쇼핑, 문화생활)
- **20%**: 저축 및 투자

### 추천 관리 방법
1. 매일 지출 기록
2. 주간 예산 확인
3. 월말 리포트 리뷰
4. 분기별 예산 조정
