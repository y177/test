magic mcp server와 ultra-mcp server의 기능을 알려줘. 

● Magic MCP Server vs Ultra-MCP Server

  Magic MCP Server

  목적: 현대적인 UI 컴포넌트 생성 및 디자인 시스템 통합

  핵심 기능

  - UI 컴포넌트 생성: React, Vue, Angular 등 프레임워크별
  컴포넌트 자동 생성
  - 디자인 시스템 통합: 기존 테마, 스타일, 토큰과 자동 통합        
  - 21st.dev 데이터베이스: 검증된 디자인 패턴 및 컴포넌트 패턴     
   라이브러리
  - 접근성 준수: WCAG 표준, 시맨틱 마크업, 키보드 네비게이션       
  자동 구현
  - 반응형 디자인: 모바일 우선 반응형 패턴 적용

  컴포넌트 카테고리

  - Interactive: 버튼, 폼, 모달, 드롭다운, 네비게이션
  - Layout: 그리드, 컨테이너, 카드, 패널, 사이드바
  - Display: 타이포그래피, 이미지, 아이콘, 차트, 테이블
  - Feedback: 알림, 진행 표시기, 툴팁, 로딩 상태
  - Input: 텍스트 필드, 날짜 선택기, 파일 업로드, 리치 에디터      

  프레임워크 지원

  - React (Hooks, TypeScript)
  - Vue (Composition API)
  - Angular (Component Architecture)
  - Vanilla (Web Components)

  자동 활성화

  - Keywords: "component", "UI", "responsive", "design"
  - Frontend persona 활성화 시
  - 디자인 시스템 작업

  ---
  Ultra-MCP Server

  목적: 멀티 AI 모델 통합 플랫폼 (OpenAI + Gemini + Grok +
  Claude)

  핵심 기능

  1. Deep Reasoning (고급 추론)

  - GPT-5, Gemini 2.5 Pro 활용
  - 복잡한 문제 해결 및 심층 분석
  - Google Search 통합 (Gemini)

  2. Investigation (조사)

  - 주제에 대한 철저한 조사
  - 깊이 설정: shallow/medium/deep
  - 웹 검색 통합

  3. Research (연구)

  - 포괄적인 연구 수행
  - 출력 형식: summary/detailed/academic
  - 다중 소스 분석

  4. Code Analysis (코드 분석)

  - 아키텍처, 성능, 보안, 품질 분석
  - 멀티 파일 분석
  - 포커스 영역 지정 가능

  5. Code Review (코드 리뷰)

  - 버그, 보안, 성능, 스타일 검토
  - 파일별 상세 분석

  6. Debug Issue (디버깅)

  - 체계적인 문제 해결
  - 근본 원인 분석
  - 증상 기반 진단

  7. Plan Feature (기능 계획)

  - 단계별 구현 계획
  - 요구사항 분석
  - 범위 설정: minimal/standard/comprehensive

  8. Generate Docs (문서 생성)

  - Markdown, API docs, README 생성
  - 다양한 형식 지원

  9. Challenge (비판적 사고)

  - 가정과 주장에 대한 비판적 분석
  - 대안 제시

  10. Consensus (합의)

  - 여러 AI 모델의 의견 수렴
  - 다양한 관점 분석

  11. Planner (계획 수립)

  - 다단계 계획 수립
  - 브랜치 및 수정 지원

  12. Pre-commit (커밋 전 검증)

  - 코드 변경 사항 검증
  - 보안, 성능, 품질 체크

  13. Security Audit (보안 감사)

  - OWASP, 규정 준수 확인
  - 취약점 분석

  14. Tracer (실행 추적)

  - 실행 흐름 추적
  - 의존성 매핑

  15. Ultra Review/Analyze/Debug/Plan/Docs

  - 단계별 워크플로우
  - 종합적인 분석

  16. Vector Search (벡터 검색)

  - 시맨틱 코드 검색
  - 프로젝트 인덱싱

  17. Session Management (세션 관리)

  - 대화 컨텍스트 유지
  - 세션 간 연속성

  18. Budget Control (예산 관리)

  - 토큰 및 비용 제어
  - 사용량 모니터링

  지원 AI 모델

  - OpenAI: GPT-4, GPT-5
  - Gemini: Gemini 2.5 Pro (Google Search 통합)
  - Grok: Grok AI
  - Azure: Azure OpenAI

  자동 활성화

  - 복잡한 분석 요청
  - 멀티 AI 모델 비교 필요 시
  - 심층 연구 및 조사

  ---
  비교 요약

  | 특징    | Magic            | Ultra-MCP            |
  |-------|------------------|----------------------|
  | 주요 목적 | UI 컴포넌트 생성       | 멀티 AI 통합 분석
        |
  | 활용 분야 | Frontend 개발      | 분석, 연구, 계획
    |
  | 강점    | 디자인 시스템, 접근성     | 심층 추론, 다중 관점     
           |
  | AI 모델 | 단일 (Claude Code) | 다중 (GPT/Gemini/Grok) |        
  | 출력    | 코드 컴포넌트          | 분석 리포트, 계획
       |

  추천 사용:
  - Magic: UI/UX 개발, 컴포넌트 생성, 디자인 작업
  - Ultra-MCP: 복잡한 분석, 전략 수립, 심층 연구, 다각도 검토
