## Description

작성된 일기를 분석하여 감정 패턴, 활동 추이, 개인 성장을 추적합니다.

## When to Use

- 주간/월간 회고 시
- 감정 패턴 파악 시
- 목표 달성도 분석 시
- 개인 성장 추적 시

## Features

### 1. 감정 분석
- 기분 점수 추이
- 감정 패턴 인식
- 스트레스 요인 식별

### 2. 활동 추적
- 생산적인 날 vs 비생산적인 날
- 목표 달성률
- 시간 사용 패턴

### 3. 인사이트 생성
- 긍정적 요인 식별
- 개선 영역 제안
- 성장 지표 추적

## Analysis Reports

### 주간 리포트
```
📊 Week of {{week}}

평균 기분: 7.2/10
목표 달성률: 65%
가장 생산적인 날: 화요일
감사 표현 횟수: 21

💡 인사이트:
- 충분한 수면을 취한 날 기분이 좋았음
- 운동한 날 생산성이 높았음
```

### 월간 리포트
```
📊 Month of {{month}}

총 일기 작성: 28/30일
평균 기분: 7.5/10
주요 감정: 행복(40%), 평온(30%)
목표 달성률: 72%

🎯 주요 달성:
- 매일 운동 챌린지 완료
- 독서 10권 완료

🔄 개선 영역:
- 늦은 취침 개선 필요
```

## Tips

- 최소 2주 데이터로 패턴 파악
- 시각화로 추이 확인
- 긍정적 변화에 집중
- 작은 성장도 기록
