## Description

일일 저널을 자동으로 생성하고, 템플릿을 적용하며, 감정과 활동을 추적합니다.

## When to Use

- 매일 아침/저녁 일기 작성 시
- 특별한 이벤트 기록 시
- 감정 및 기분 추적 시
- 회고 및 반성 시

## How to Use

1. 오늘 날짜로 저널 파일 생성
2. 템플릿 적용 (아침/저녁/자유 형식)
3. 프롬프트에 따라 작성 안내
4. 자동 태그 및 메타데이터 추가

## Journal Templates

### 아침 저널
```markdown
# 🌅 {{date}} 아침

## 😊 오늘의 기분
[1-10 점수]

## 🎯 오늘의 목표
- [ ]
- [ ]
- [ ]

## 💭 아침 생각
```

### 저녁 저널
```markdown
# 🌙 {{date}} 저녁

## 📝 오늘의 하이라이트
-

## ✅ 목표 달성도
- [x]
- [ ]

## 🙏 감사한 일
1.
2.
3.

## 💡 배운 점

## 🔄 개선할 점
```

### 자유 형식
```markdown
# 📔 {{date}}

```

## Best Practices

- 매일 같은 시간에 작성
- 솔직하게 감정 표현
- 구체적으로 기록
- 정기적으로 과거 일기 리뷰
- 개인정보 보호 유의
