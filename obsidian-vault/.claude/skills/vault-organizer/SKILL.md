## Description

노트를 분류하고, 태그를 정리하며, 깨진 링크를 수정하고, vault 구조를 최적화합니다.

## When to Use

- Vault 구조 개선이 필요할 때
- 깨진 링크를 찾아 수정할 때
- 태그 체계를 정리할 때
- 중복 노트를 병합할 때

## Features

### 1. 링크 관리
- 깨진 링크 탐지 및 수정
- 미사용 노트 찾기
- 링크 그래프 분석

### 2. 태그 정리
- 중복/유사 태그 병합
- 태그 계층 구조 재구성
- 미사용 태그 제거

### 3. 파일 구조
- 폴더 구조 최적화
- 파일명 일괄 변경
- 아카이브 관리

## Maintenance Tasks

```bash
# 주간 정리
- 깨진 링크 확인
- 고아 노트(orphan notes) 처리
- 태그 정리

# 월간 정리
- Vault 백업
- 구조 최적화
- 통계 리포트 생성
```

## Tips

- 정기적인 vault 백업 필수
- Graph view로 링크 구조 시각화
- Dataview 플러그인과 연계하여 자동 인덱스 생성
