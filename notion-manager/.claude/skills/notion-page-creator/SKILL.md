## Description

이 스킬은 Notion API를 사용하여 페이지를 생성하고, 데이터베이스 항목을 추가하며, 콘텐츠를 자동으로 관리합니다.

## When to Use

- 사용자가 Notion 페이지 생성을 요청할 때
- Notion 데이터베이스에 항목을 추가해야 할 때
- 템플릿 기반으로 페이지를 생성할 때
- 로컬 파일을 Notion으로 동기화할 때

## How to Use

1. Notion MCP 서버가 구성되어 있는지 확인
2. 사용자의 요청에 따라 페이지 구조 설계
3. Notion API를 통해 페이지 생성
4. 생성된 페이지 URL 반환

## Examples

### 예제 1: 간단한 페이지 생성
사용자: "오늘의 회의록 페이지를 만들어줘"
- 제목: "[날짜] 회의록"
- 기본 구조: 참석자, 안건, 논의 내용, 액션 아이템

### 예제 2: 데이터베이스 항목 추가
사용자: "독서 목록에 새 책 추가해줘"
- 책 제목, 저자, 상태, 시작일 등의 속성 설정

## Tips

- 템플릿을 미리 정의하여 재사용성 높이기
- 페이지 생성 후 URL을 항상 사용자에게 제공
- 에러 발생 시 명확한 에러 메시지 제공
